# García Sellerie Garniture - PRD

## Problem Statement
Recréer le site web García Sellerie Garniture basé sur le template craft-upholstery, avec le logo fourni par l'utilisateur.

## Architecture
- **Frontend**: React.js avec Tailwind CSS
- **Backend**: FastAPI (Python)
- **Database**: MongoDB
- **Styling**: Custom CSS avec animations

## User Personas
- Clients particuliers (automobile, moto)
- Professionnels du nautisme
- Designers d'intérieur (mobilier)

## Core Requirements
- Site vitrine moderne et élégant
- Présentation des 4 domaines d'expertise
- Galerie de réalisations avec filtres
- Formulaire de contact fonctionnel
- Navigation fluide avec smooth scroll
